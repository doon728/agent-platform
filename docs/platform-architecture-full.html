Enterprise AI Agent Platform — Full System Architecture

Four logical layers: L1 Control Plane (Agent Factory UI, Support API, Tool Admin UI) · L2 Agent Runtime — C1 (per-customer, per-agent — deployed wherever the customer wants) · L3 Reasoning Engine — C2 + MCP + C3 Tool Gateway (platform-managed) · L4 Data (customer-managed, never leaves customer environment). Shown here as a two-VPC split — one common deployment option. Can also run as a single VPC, on-prem, or cloud-agnostic. C3 calls customer systems over HTTPS only — customer controls what is exposed. LLM API is external.
